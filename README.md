# Skin-pack-Template
Minecraft LCE Skin-pack Template

# Documentation
`.pck` is the primary file extension you will find  

---

# Needed tools

- https://github.com/LCERD/PCK-Studio to edit `.pck` files
- https://libresprite.github.io/#!/ : image editor
- https://minecraft.tools/en/skin.php to "steal" skins

Feel free to use https://www.virustotal.com/gui/home/upload if you want to scan those tools.  
If you don't want to use the provided tools, you can search other ones, there are plenty out there :)

# Guide

Use https://minecraft.tools/en/skin.php to get any minecraft skin with a username.



After editing your .pck skin file you need to put it in a folder.
Then, you need to move your folder to "LCEWindows64\Windows64Media\DLC".

I recommend naming your folder the same as your .pck file.

Recommended tools:
- .pck editor: https://github.com/LCERD/PCK-Studio
- image editor: https://libresprite.github.io/#!/
- Skin "stealer": https://minecraft.tools/en/skin.php

Tutorial: https://www.youtube.com/watch?v=Tme_ywgBoDs
